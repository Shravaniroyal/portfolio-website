Project Overview

Advanced Security Solution

This innovative project addresses critical security challenges in biometric data protection through the integration of Rubik's Cube-based encryption with cutting-edge biometric authentication technologies. The system leverages iris and fingerprint authentication powered by Convolutional Neural Networks (CNNs) for superior accuracy and reliability.

Key Innovations

  • Zero-bit watermarking for data integrity
  • Rubik's Cube encryption algorithm
  • Dual biometric authentication (Iris + Fingerprint)
  • CNN-powered feature extraction
